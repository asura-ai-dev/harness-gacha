# Task 4002: Entitlement データ読み書き

## 概要

購入済み pack の権利情報（entitlement）を JSON ファイルから読み書きする層を `src/data/entitlement.rs` に実装する。サンプルデータも作成する。

## 詳細

### 1. `data/entitlements.json` の作成

`agent-docs/data-models.md` の JSON ファイル構造に従い、サンプルデータを作成する。

```json
{
  "user_id": "local-user",
  "entitlements": [
    {
      "pack_id": "team-review-pack",
      "purchased_at": "2026-04-05T10:00:00Z",
      "version_at_purchase": "1.0.0",
      "status": "active",
      "installed": true,
      "installed_version": "1.0.0"
    },
    {
      "pack_id": "frontend-spec-kit",
      "purchased_at": "2026-04-08T14:30:00Z",
      "version_at_purchase": "1.0.0",
      "status": "active",
      "installed": false
    }
  ]
}
```

最低 2 件の entitlement を含める。1 件は installed=true、もう 1 件は installed=false にする。

### 2. `src/data/entitlement.rs`

```rust
use std::path::Path;
use crate::models::entitlement::{EntitlementStore, EntitlementEntry};

/// entitlements.json を読み込み、EntitlementStore を返す
/// ファイルが存在しない場合はデフォルト値（空の entitlements）を返す
pub fn load_entitlements(path: &Path) -> EntitlementStore {
    match std::fs::read_to_string(path) {
        Ok(content) => {
            serde_json::from_str::<EntitlementStore>(&content).unwrap_or_else(|e| {
                eprintln!("entitlements.json パースエラー: {}", e);
                default_store()
            })
        }
        Err(_) => default_store(),
    }
}

/// entitlements.json に書き込む
pub fn save_entitlements(path: &Path, store: &EntitlementStore) -> Result<(), String> {
    let json = serde_json::to_string_pretty(store)
        .map_err(|e| format!("JSON シリアライズエラー: {}", e))?;
    std::fs::write(path, json)
        .map_err(|e| format!("ファイル書き込みエラー: {}", e))
}

/// デフォルトの空 EntitlementStore を返す
fn default_store() -> EntitlementStore {
    EntitlementStore {
        user_id: "local-user".to_string(),
        entitlements: Vec::new(),
    }
}

/// active な entitlement のみを返す
pub fn active_entitlements(store: &EntitlementStore) -> Vec<&EntitlementEntry> {
    store.entitlements.iter()
        .filter(|e| e.status == "active")
        .collect()
}

/// 指定 pack_id の active な entitlement が存在するか
pub fn is_owned(store: &EntitlementStore, pack_id: &str) -> bool {
    store.entitlements.iter()
        .any(|e| e.pack_id == pack_id && e.status == "active")
}

/// 指定 pack_id の entitlement を検索して返す
pub fn find_entitlement<'a>(store: &'a EntitlementStore, pack_id: &str) -> Option<&'a EntitlementEntry> {
    store.entitlements.iter()
        .find(|e| e.pack_id == pack_id)
}
```

### 3. `src/app.rs` の更新

`App::new()` メソッドで entitlements.json も読み込むようにする。

```rust
// App::new() 内で追加
let entitlements = crate::data::entitlement::load_entitlements(&data_dir.join("entitlements.json"));
// entitlements フィールドを Option<EntitlementStore> から EntitlementStore に変更するか、
// Some(entitlements) を設定する
```

`App` struct の `entitlements` フィールドの型を `Option<EntitlementStore>` から `EntitlementStore` に変更することを推奨する。load_entitlements が常にデフォルト値を返すため None になることがない。ただし既存コードとの互換を考慮し、`Option` のまま `Some(entitlements)` を設定してもよい。

### 4. テストの追加

`src/data/entitlement.rs` にテストを追加する。

```rust
#[cfg(test)]
mod tests {
    use super::*;

    fn sample_store() -> EntitlementStore {
        EntitlementStore {
            user_id: "local-user".to_string(),
            entitlements: vec![
                EntitlementEntry {
                    pack_id: "pack-a".to_string(),
                    purchased_at: "2026-04-01T00:00:00Z".to_string(),
                    version_at_purchase: "1.0.0".to_string(),
                    status: "active".to_string(),
                    installed: true,
                    installed_version: Some("1.0.0".to_string()),
                },
                EntitlementEntry {
                    pack_id: "pack-b".to_string(),
                    purchased_at: "2026-04-02T00:00:00Z".to_string(),
                    version_at_purchase: "1.0.0".to_string(),
                    status: "refunded".to_string(),
                    installed: false,
                    installed_version: None,
                },
            ],
        }
    }

    #[test]
    fn test_active_entitlements() {
        let store = sample_store();
        let active = active_entitlements(&store);
        assert_eq!(active.len(), 1);
        assert_eq!(active[0].pack_id, "pack-a");
    }

    #[test]
    fn test_is_owned() {
        let store = sample_store();
        assert!(is_owned(&store, "pack-a"));
        assert!(!is_owned(&store, "pack-b")); // refunded
        assert!(!is_owned(&store, "pack-c")); // 存在しない
    }

    #[test]
    fn test_find_entitlement() {
        let store = sample_store();
        assert!(find_entitlement(&store, "pack-a").is_some());
        assert!(find_entitlement(&store, "nonexistent").is_none());
    }
}
```
