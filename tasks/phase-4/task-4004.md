# Task 4004: インストール詳細画面

## 概要

購入済み pack のインストール手順を表示する画面を `src/ui/install_detail.rs` に実装する。F-7（インストール案内）に対応。

## 詳細

### 1. `src/ui/install_detail.rs`

`agent-docs/ui-components.md` の画面 7（インストール詳細）に従い実装する。

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 3 行固定 - "Install: {pack_name}"
2. コンテンツ: 残り全部 - method、command、post_install、rollback
3. フッター: 3 行固定 - キーバインド

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Wrap};
use crate::app::App;
use crate::ui::theme::Theme;
use crate::data::catalog::find_pack_by_id;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    let pack = match &app.selected_pack_id {
        Some(id) => find_pack_by_id(&app.catalog, id),
        None => None,
    };
    let pack = match pack {
        Some(p) => p,
        None => {
            let msg = Paragraph::new("Pack not found")
                .style(theme.text_style())
                .alignment(Alignment::Center);
            frame.render_widget(msg, frame.area());
            return;
        }
    };

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    // ヘッダー
    let header = Paragraph::new(format!("Install: {}", pack.name))
        .style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()).style(theme.panel_style()));
    frame.render_widget(header, chunks[0]);

    // コンテンツ
    render_content(frame, chunks[1], pack, app.scroll_offset, theme);

    // フッター
    let footer = Paragraph::new("[j/k] スクロール  [b] 戻る")
        .style(theme.footer_text_style())
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()));
    frame.render_widget(footer, chunks[2]);
}

fn render_content(
    frame: &mut Frame,
    area: Rect,
    pack: &crate::models::catalog_entry::CatalogEntry,
    scroll: u16,
    theme: &Theme,
) {
    let mut lines: Vec<Line> = Vec::new();

    // Method
    lines.push(Line::from(Span::styled(
        "Method",
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));
    lines.push(Line::from(Span::styled(
        format!("  {}", pack.install.method),
        Style::default().fg(theme.text_primary),
    )));
    lines.push(Line::from(""));

    // Entrypoint（あれば）
    if let Some(ep) = &pack.install.entrypoint {
        lines.push(Line::from(Span::styled(
            "Entrypoint",
            Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
        )));
        lines.push(Line::from(Span::styled(
            format!("  {}", ep),
            Style::default().fg(theme.text_primary),
        )));
        lines.push(Line::from(""));
    }

    // Command / Steps
    if let Some(steps) = &pack.install.steps {
        lines.push(Line::from(Span::styled(
            "Command",
            Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
        )));
        for step in steps {
            lines.push(Line::from(Span::styled(
                format!("  $ {}", step),
                Style::default().fg(theme.owned), // コマンドは owned 色（青系）で目立たせる
            )));
        }
        lines.push(Line::from(""));
    }

    // Post Install
    // CatalogEntry には post_install フィールドがないため、固定テキストで案内する
    // 将来的には Manifest から読み込む
    lines.push(Line::from(Span::styled(
        "Post Install",
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));
    lines.push(Line::from(Span::styled(
        "  - 設定ファイルを確認してください",
        Style::default().fg(theme.text_secondary),
    )));
    lines.push(Line::from(Span::styled(
        "  - 必要に応じて hooks を有効化してください",
        Style::default().fg(theme.text_secondary),
    )));
    lines.push(Line::from(""));

    // Rollback
    lines.push(Line::from(Span::styled(
        "Rollback",
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));
    lines.push(Line::from(Span::styled(
        "  - 追加ファイルを削除してください",
        Style::default().fg(theme.text_secondary),
    )));
    lines.push(Line::from(Span::styled(
        "  - 変更前のバックアップを復元してください",
        Style::default().fg(theme.text_secondary),
    )));

    let block = Block::default()
        .borders(Borders::LEFT | Borders::RIGHT)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(lines)
        .block(block)
        .wrap(Wrap { trim: false })
        .scroll((scroll, 0));
    frame.render_widget(p, area);
}
```

### 2. `src/ui/render.rs` の更新

`Screen::InstallDetail` のプレースホルダーを `install_detail::render(frame, app, &theme)` に置き換える。

### 3. `src/app.rs` の更新

`handle_screen_action` に `Screen::InstallDetail` のハンドラを追加する。

```rust
Screen::InstallDetail => {
    match action {
        Action::Up => {
            self.scroll_offset = self.scroll_offset.saturating_sub(1);
        }
        Action::Down => {
            self.scroll_offset = self.scroll_offset.saturating_add(1);
        }
        _ => {}
    }
}
```
