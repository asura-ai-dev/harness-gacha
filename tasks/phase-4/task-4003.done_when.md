# Task 4003 done_when

- `cargo build` が成功する
- カタログ画面で `L` キーを押すと購入済みライブラリ画面に遷移する
- ライブラリ画面のヘッダーに "My Library" が表示される
- `data/entitlements.json` に記載された active な pack が一覧に表示される
- 各行に pack 名、バージョン、install 状態（Installed/Not Installed）、update 状態が表示される
- installed=true の pack が owned 色（青系）で "Installed" と表示される
- installed=false の pack が text_secondary 色で "Not Installed" と表示される
- installed_version と catalog の version が異なる場合に "Update: available" が warning 色で表示される
- status="refunded" の entitlement がリストに表示されない
- entitlements が空の場合に "購入済み pack はありません" が表示されクラッシュしない
- `j`/`Down` と `k`/`Up` キーでリスト内の選択行が移動する
- `Enter` キーで選択中の pack のインストール詳細画面に遷移する（プレースホルダー表示でよい）
- `b` または `Esc` でカタログ画面に戻る
