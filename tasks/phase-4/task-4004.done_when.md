# Task 4004 done_when

- `cargo build` が成功する
- 購入済みライブラリ画面で pack を選択して `Enter` を押すとインストール詳細画面に遷移する
- インストール詳細画面のヘッダーに "Install: {pack名}" が表示される
- Method セクションに install.method（"script", "copy", "manual" のいずれか）が表示される
- install.entrypoint がある pack で Entrypoint セクションが表示される
- install.steps がある pack で Command セクションに各ステップが "$ {step}" 形式で表示される
- コマンドが owned 色（青系）で表示される
- Post Install セクションに導入後の案内テキストが表示される
- Rollback セクションに rollback 手順のテキストが表示される
- `j`/`Down` と `k`/`Up` でコンテンツがスクロールする
- `b` または `Esc` で購入済みライブラリ画面に戻る
- selected_pack_id が None の場合に "Pack not found" が表示されクラッシュしない
- `Screen::InstallDetail` が render.rs のディスパッチに接続されている
