# Task 4001: 購入案内画面（QR コード・ブラウザ起動・URL コピー）

## 概要

購入案内画面を `src/ui/purchase.rs` に実装する。QR コード Widget、ブラウザ起動、URL クリップボードコピーを含む。F-5（購入案内と Web checkout 遷移）に対応。

## 詳細

### 1. `src/ui/widgets/qr_code.rs`

URL 文字列からテキスト QR コードを生成するヘルパー。

```rust
use ratatui::prelude::*;
use ratatui::text::Line;

/// URL 文字列から QR コードのテキスト表現を Line ベクタとして返す
/// qrcode クレートで QR を生成し、ブロック文字で描画する
/// エラー時は "QR generation failed" を返す
pub fn generate_qr_lines(url: &str, fg: Color, bg: Color) -> Vec<Line<'static>> {
    use qrcode::QrCode;

    let code = match QrCode::new(url.as_bytes()) {
        Ok(c) => c,
        Err(_) => {
            return vec![Line::from(Span::styled(
                "QR generation failed".to_string(),
                Style::default().fg(fg),
            ))];
        }
    };

    let string = code
        .render::<char>()
        .quiet_zone(false)
        .module_dimensions(2, 1)
        .build();

    string
        .lines()
        .map(|line| {
            Line::from(Span::styled(
                line.to_string(),
                Style::default().fg(fg).bg(bg),
            ))
        })
        .collect()
}
```

### 2. `src/browser.rs`

```rust
/// URL をデフォルトブラウザで開く
/// 成功時は Ok(()), 失敗時はエラーメッセージを返す
pub fn open_url(url: &str) -> Result<(), String> {
    open::that(url).map_err(|e| format!("ブラウザの起動に失敗しました: {}", e))
}
```

### 3. `src/clipboard.rs`

```rust
/// テキストをシステムクリップボードにコピーする
/// 成功時は Ok(()), 失敗時はエラーメッセージを返す
pub fn copy_to_clipboard(text: &str) -> Result<(), String> {
    use arboard::Clipboard;
    let mut clipboard = Clipboard::new()
        .map_err(|e| format!("クリップボードの初期化に失敗しました: {}", e))?;
    clipboard
        .set_text(text.to_string())
        .map_err(|e| format!("クリップボードへのコピーに失敗しました: {}", e))
}
```

### 4. `src/ui/purchase.rs`

`agent-docs/ui-components.md` の画面 5（購入案内）に従い実装する。

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 3 行固定 - "Checkout: {pack_name}"
2. コンテンツ: 残り全部 - price, seller, policy, URL, QR
3. フッター: 3 行固定 - キーバインド

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Wrap};
use crate::app::App;
use crate::ui::theme::Theme;
use crate::ui::widgets::qr_code;
use crate::data::catalog::find_pack_by_id;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    let pack = match &app.selected_pack_id {
        Some(id) => find_pack_by_id(&app.catalog, id),
        None => None,
    };
    let pack = match pack {
        Some(p) => p,
        None => {
            let msg = Paragraph::new("Pack not found")
                .style(theme.text_style()).alignment(Alignment::Center);
            frame.render_widget(msg, frame.area());
            return;
        }
    };

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    // ヘッダー
    let header = Paragraph::new(format!("Checkout: {}", pack.name))
        .style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()).style(theme.panel_style()));
    frame.render_widget(header, chunks[0]);

    // コンテンツ
    render_content(frame, chunks[1], pack, theme);

    // フッター
    let footer_text = if app.message.is_some() {
        app.message.as_deref().unwrap_or("").to_string()
    } else {
        "[Enter] Checkout を開く  [c] URL コピー  [b] 戻る".to_string()
    };
    let footer = Paragraph::new(footer_text)
        .style(theme.footer_text_style())
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()));
    frame.render_widget(footer, chunks[2]);
}

fn render_content(frame: &mut Frame, area: Rect, pack: &CatalogEntry, theme: &Theme) {
    // 左右 2 分割: 左に情報テキスト、右に QR コード
    let h_chunks = Layout::default()
        .direction(Direction::Horizontal)
        .constraints([
            Constraint::Percentage(55),
            Constraint::Percentage(45),
        ])
        .split(area);

    // 左: テキスト情報
    let mut lines: Vec<Line> = Vec::new();

    // Price
    lines.push(Line::from(Span::styled("Price", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    lines.push(Line::from(Span::styled(format!("  YEN {}", pack.price), Style::default().fg(theme.accent_alt))));
    lines.push(Line::from(""));

    // Seller
    lines.push(Line::from(Span::styled("Seller", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    lines.push(Line::from(Span::styled("  harness-gacha", Style::default().fg(theme.text_primary))));
    lines.push(Line::from(""));

    // Policy
    lines.push(Line::from(Span::styled("Policy", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    lines.push(Line::from(Span::styled("  デジタル商品のため購入後の返金は条件付き", Style::default().fg(theme.text_secondary))));
    lines.push(Line::from(Span::styled("  利用条件は Web の規約を参照", Style::default().fg(theme.text_secondary))));
    lines.push(Line::from(""));

    // Checkout URL
    lines.push(Line::from(Span::styled("Checkout URL", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    lines.push(Line::from(Span::styled(format!("  {}", pack.checkout_url), Style::default().fg(theme.owned))));

    let left_block = Block::default()
        .borders(Borders::LEFT | Borders::RIGHT)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let left = Paragraph::new(lines).block(left_block).wrap(Wrap { trim: false });
    frame.render_widget(left, h_chunks[0]);

    // 右: QR コード
    let mut qr_lines: Vec<Line> = Vec::new();
    qr_lines.push(Line::from(Span::styled("QR Code", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    qr_lines.push(Line::from(""));
    qr_lines.extend(qr_code::generate_qr_lines(&pack.checkout_url, theme.text_primary, theme.panel_bg));

    let right_block = Block::default()
        .borders(Borders::RIGHT)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let right = Paragraph::new(qr_lines).block(right_block);
    frame.render_widget(right, h_chunks[1]);
}
```

### 6. `src/app.rs` の更新

`handle_screen_action` に `Screen::Purchase` のハンドラを追加する。

```rust
Screen::Purchase => {
    match action {
        Action::Enter => {
            // ブラウザで checkout URL を開く
            if let Some(id) = &self.selected_pack_id {
                if let Some(pack) = crate::data::catalog::find_pack_by_id(&self.catalog, id) {
                    match crate::browser::open_url(&pack.checkout_url) {
                        Ok(_) => self.message = Some("ブラウザを起動しました".to_string()),
                        Err(e) => self.message = Some(e),
                    }
                }
            }
        }
        Action::CopyUrl => {
            // URL をクリップボードにコピー
            if let Some(id) = &self.selected_pack_id {
                if let Some(pack) = crate::data::catalog::find_pack_by_id(&self.catalog, id) {
                    match crate::clipboard::copy_to_clipboard(&pack.checkout_url) {
                        Ok(_) => self.message = Some("URL をコピーしました".to_string()),
                        Err(e) => self.message = Some(e),
                    }
                }
            }
        }
        _ => {}
    }
}
```

また、`Action::Tick` の処理で、`self.message` を一定 tick 後にクリアするロジックを追加する（例: message 設定後 8 tick = 2 秒で None に戻す）。これは `message_clear_at: Option<u64>` フィールドを App に追加して管理する。
