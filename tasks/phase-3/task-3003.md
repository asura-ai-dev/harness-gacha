# Task 3003: Discovery 状態遷移と演出アニメーション接続

## 概要

Discovery の待機 -> 演出 -> 結果の状態遷移を `src/app.rs` に接続する。tick イベントで演出フレームを進め、完了後に結果状態に遷移する。結果から Pack 詳細画面への遷移も実装する。

## 詳細

### 1. `src/app.rs` の更新

#### Discovery 画面のアクションハンドラを追加

`handle_screen_action` メソッド内の `Screen::Discovery` 分岐を実装する。

```rust
Screen::Discovery => {
    match &self.discovery_state {
        DiscoveryState::Idle => {
            match action {
                Action::Enter => {
                    // ランダム pack を選択して演出開始
                    if let Some(pack_id) = crate::discovery::pick_random_pack(&self.catalog) {
                        self.discovery_state = DiscoveryState::Animating {
                            frame: 0,
                            target_pack_id: pack_id,
                        };
                    }
                }
                _ => {}
            }
        }
        DiscoveryState::Animating { .. } => {
            // 演出中は Enter で skip 可能
            match action {
                Action::Enter => {
                    // skip: 即座に結果状態へ
                    if let DiscoveryState::Animating { target_pack_id, .. } = &self.discovery_state {
                        let id = target_pack_id.clone();
                        self.discovery_state = DiscoveryState::Result { pack_id: id };
                    }
                }
                _ => {}
            }
        }
        DiscoveryState::Result { pack_id } => {
            match action {
                Action::Enter => {
                    // 結果から Pack 詳細へ遷移
                    if let DiscoveryState::Result { pack_id } = &self.discovery_state {
                        self.selected_pack_id = Some(pack_id.clone());
                        self.navigate_to(Screen::PackDetail);
                    }
                }
                Action::ToggleDiscovery => {
                    // もう一度探す: 新しい pack を選んで演出開始
                    if let Some(new_id) = crate::discovery::pick_random_pack(&self.catalog) {
                        self.discovery_state = DiscoveryState::Animating {
                            frame: 0,
                            target_pack_id: new_id,
                        };
                    }
                }
                _ => {}
            }
        }
    }
}
```

#### Back の特殊処理

Discovery 画面で `Back` が来た場合、discovery_state を Idle にリセットしてから戻る。`App::update` メソッドの `Action::Back` 処理を拡張する。

```rust
Action::Back => {
    if self.current_screen == Screen::Discovery {
        self.discovery_state = DiscoveryState::Idle;
    }
    self.go_back();
}
```

#### Tick でアニメーションフレームを進める

`App::update` メソッドの `Action::Tick` 処理を拡張する。

```rust
Action::Tick => {
    self.tick_count += 1;
    // Discovery 演出中ならフレームを進める
    if let DiscoveryState::Animating { frame, target_pack_id } = &self.discovery_state {
        let next_frame = frame + 1;
        if next_frame >= crate::ui::widgets::capsule_machine::TOTAL_FRAMES {
            // 演出完了 → 結果状態へ
            let id = target_pack_id.clone();
            self.discovery_state = DiscoveryState::Result { pack_id: id };
        } else {
            let id = target_pack_id.clone();
            self.discovery_state = DiscoveryState::Animating {
                frame: next_frame,
                target_pack_id: id,
            };
        }
    }
}
```

### 2. Discovery からカタログへ遷移する際の状態リセット

カタログ画面から Discovery に遷移する際に `discovery_state` を `Idle` に初期化する。`handle_catalog_action` の `Action::ToggleDiscovery` 分岐を更新する。

```rust
Action::ToggleDiscovery => {
    self.discovery_state = DiscoveryState::Idle;
    self.navigate_to(Screen::Discovery);
}
```

### 3. 演出中の Back 無効化

演出中（Animating 状態）の場合、Back を無視するか、Enter での skip のみ許可する。`App::update` 内の `Action::Back` 判定で、Discovery の Animating 中なら何もしないようにする。

```rust
Action::Back => {
    // 演出中は Back を無視
    if self.current_screen == Screen::Discovery {
        if let DiscoveryState::Animating { .. } = &self.discovery_state {
            return; // 何もしない
        }
        self.discovery_state = DiscoveryState::Idle;
    }
    self.go_back();
}
```
