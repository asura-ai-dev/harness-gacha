# Task 3001: Discovery ロジックとカプセルマシン Widget

## 概要

ランダム pack 選択ロジックを `src/discovery.rs` に、ASCII アートのガチャガチャマシンを `src/ui/widgets/capsule_machine.rs` に実装する。

## 詳細

### 1. `src/discovery.rs`

カタログからランダムに 1 pack を選択するロジックを実装する。

```rust
use rand::seq::SliceRandom;
use crate::models::CatalogEntry;

/// status == "listed" の pack からランダムに 1 つ選び、その id を返す
/// カタログが空の場合は None を返す
pub fn pick_random_pack(catalog: &[CatalogEntry]) -> Option<String> {
    let listed: Vec<&CatalogEntry> = catalog.iter()
        .filter(|p| p.status == "listed")
        .collect();
    let mut rng = rand::thread_rng();
    listed.choose(&mut rng).map(|p| p.id.clone())
}
```

テストを追加:


### 2. `src/ui/widgets/capsule_machine.rs`

ASCII アートのガチャガチャマシンを描画する。フレーム番号（0-5）に応じた異なる ASCII アートを返す。

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Paragraph, Widget};
use crate::ui::theme::Theme;

/// フレーム番号に応じたガチャガチャマシンの ASCII アート文字列を返す
/// frame: 0-5（6 フレーム、各 250ms で約 1.5 秒）
pub fn capsule_art(frame: u8) -> Vec<&'static str> {
    match frame {
        // 待機状態（frame 引数は使わないが、Idle 表示用に 255 等を渡す想定）
        0 => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○  ○  ○    │",
            "│        ▒▒        │",
            "│       ─┼─        │",
            "└──────────────────┘",
        ],
        // Frame 1: レバー回転開始
        1 => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○  ○  ○    │",
            "│        ▒▒        │",
            "│       \\│/        │",
            "└──────────────────┘",
        ],
        // Frame 2: レバー回転完了、カプセル移動開始
        2 => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○     ○    │",
            "│        ▒▒        │",
            "│       ─┼─    ○   │",
            "└──────────────────┘",
        ],
        // Frame 3: カプセル落下中
        3 => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○     ○    │",
            "│                  │",
            "│       ─┼─        │",
            "└────────○─────────┘",
        ],
        // Frame 4: カプセル開封中
        4 => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○     ○    │",
            "│                  │",
            "│       ─┼─        │",
            "└──────────────────┘",
            "",
            "        * ○ *        ",
        ],
        // Frame 5: 開封完了
        _ => vec![
            "┌──────────────────┐",
            "│   CAPSULE GAME   │",
            "│                  │",
            "│      ○  ○  ○     │",
            "│    ○  ○     ○    │",
            "│                  │",
            "│       ─┼─        │",
            "└──────────────────┘",
            "",
            "     [ OPENED! ]     ",
        ],
    }
}

/// capsule_art を ratatui の Line のベクタに変換する
/// color にはフレームに応じた色を渡す
pub fn capsule_lines(frame: u8, theme: &Theme) -> Vec<Line<'static>> {
    let art = capsule_art(frame);
    let color = match frame {
        4 | 5 => theme.accent_alt,
        _ => theme.border,
    };
    art.into_iter()
        .map(|line| {
            Line::from(Span::styled(line.to_string(), Style::default().fg(color)))
        })
        .collect()
}

/// 待機状態のマシンアートを返す
pub fn idle_art(theme: &Theme) -> Vec<Line<'static>> {
    capsule_lines(0, theme)
}

/// アニメーションの総フレーム数
pub const TOTAL_FRAMES: u8 = 6;
```

### 3. `src/ui/widgets/pack_card.rs`

Discovery 結果画面で使用する pack カード Widget を実装する。

```rust
use ratatui::prelude::*;
use ratatui::text::{Line, Span};
use crate::models::CatalogEntry;
use crate::ui::theme::Theme;

/// CatalogEntry から Discovery 結果カード用の Line ベクタを生成する
pub fn pack_card_lines<'a>(pack: &CatalogEntry, theme: &Theme) -> Vec<Line<'a>> {
    let mut lines = Vec::new();

    // Pack 名（accent, bold）
    lines.push(Line::from(Span::styled(
        pack.name.clone(),
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));

    // Summary
    lines.push(Line::from(Span::styled(
        pack.summary.clone(),
        Style::default().fg(theme.text_primary),
    )));
    lines.push(Line::from(""));

    // Creator
    lines.push(Line::from(Span::styled(
        format!("Creator: {}", pack.author.name),
        Style::default().fg(theme.text_secondary),
    )));

    // Targets
    let targets_str: String = pack.targets.iter()
        .map(|t| format!("{} {}", t.tool, t.version_range))
        .collect::<Vec<_>>()
        .join(" / ");
    lines.push(Line::from(Span::styled(
        format!("Targets: {}", targets_str),
        Style::default().fg(theme.text_secondary),
    )));

    // Price
    lines.push(Line::from(Span::styled(
        format!("Price: YEN {}", pack.price),
        Style::default().fg(theme.accent_alt),
    )));

    // Trust summary (1 行)
    let trust = if pack.permissions.has_danger() {
        Span::styled("Risk: High", Style::default().fg(theme.danger))
    } else {
        Span::styled("Safe: Low", Style::default().fg(theme.success))
    };
    lines.push(Line::from(vec![
        Span::styled("Trust: ", Style::default().fg(theme.text_secondary)),
        trust,
    ]));

    // Contents summary
    let cs = &pack.contents_summary;
    lines.push(Line::from(Span::styled(
        format!("skills {}件 / hooks {}件 / templates {}件", cs.skills, cs.hooks, cs.templates),
        Style::default().fg(theme.text_secondary),
    )));

    lines
}
```
