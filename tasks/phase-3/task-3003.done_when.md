# Task 3003 done_when

- Discovery 待機状態で `Enter` を押すと演出状態に遷移する
- 演出状態でフレームが tick ごとに自動で進む（約 250ms 間隔で 6 フレーム、計約 1.5 秒）
- 演出中にマシン ASCII アートが変化する（レバー操作 -> カプセル落下 -> 開封の流れ）
- 演出が 6 フレーム完了すると自動的に結果状態に遷移する
- 演出中に `Enter` を押すと演出をスキップして即座に結果状態に遷移する
- 演出中に `b` や `Esc` を押しても画面が戻らない（演出中は Back 無効）
- 結果状態で pack 名、summary、creator、targets、price、trust summary が表示される
- 結果状態で `Enter` を押すと Pack 詳細画面に遷移する（selected_pack_id が設定される）
- 結果状態で `r` を押すと新しいランダム pack で演出が再開する
- 結果状態で `b` を押すとカタログ画面に戻り、discovery_state が Idle にリセットされる
- カタログが空の場合に Discovery で `Enter` を押してもクラッシュしない
- Discovery は無料で実行でき、課金に関連する処理が一切含まれない
