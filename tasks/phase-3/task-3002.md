# Task 3002: Discovery 画面描画（3 状態）

## 概要

Discovery 画面の 3 状態（待機・演出・結果）の描画を `src/ui/discovery.rs` に実装する。F-2（無料ランダム Discovery）に対応。

## 詳細

### 1. `src/ui/discovery.rs`

`agent-docs/ui-components.md` の画面 2（Discovery）に従い実装する。`app.discovery_state` に応じて 3 つの描画関数を切り替える。

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph};
use crate::app::{App, DiscoveryState};
use crate::ui::theme::Theme;
use crate::ui::widgets::{capsule_machine, pack_card};
use crate::data::catalog::find_pack_by_id;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    match &app.discovery_state {
        DiscoveryState::Idle => render_idle(frame, theme),
        DiscoveryState::Animating { frame: anim_frame, .. } => {
            render_animating(frame, *anim_frame, theme);
        }
        DiscoveryState::Result { pack_id } => {
            let pack = find_pack_by_id(&app.catalog, pack_id);
            render_result(frame, pack, theme);
        }
    }
}
```

#### 待機状態 (`render_idle`)

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 3 行固定 - "Capsule Discovery"
2. コンテンツ: 残り全部 - マシン ASCII アート + CTA テキスト
3. フッター: 3 行固定 - キーバインド

コンテンツ部分は中央配置。`capsule_machine::idle_art(theme)` でマシンを描画し、その下に空行、"PRESS ENTER TO DISCOVER"（`theme.accent` 色、BOLD）、"無料でおすすめ pack を 1 つ表示します"（`theme.text_secondary` 色）を表示。

```rust
fn render_idle(frame: &mut Frame, theme: &Theme) {
    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    // ヘッダー
    let header = Paragraph::new("Capsule Discovery")
        .style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()).style(theme.panel_style()));
    frame.render_widget(header, chunks[0]);

    // コンテンツ: マシンアート + CTA
    let mut lines = capsule_machine::idle_art(theme);
    lines.push(Line::from(""));
    lines.push(Line::from(Span::styled(
        "PRESS ENTER TO DISCOVER",
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));
    lines.push(Line::from(Span::styled(
        "無料でおすすめ pack を 1 つ表示します",
        Style::default().fg(theme.text_secondary),
    )));
    let content = Paragraph::new(lines)
        .alignment(Alignment::Center)
        .block(Block::default().borders(Borders::LEFT | Borders::RIGHT).border_style(theme.border_style()).style(theme.panel_style()));
    frame.render_widget(content, chunks[1]);

    // フッター
    let footer = Paragraph::new("[Enter] ランダムで探す  [b] 戻る")
        .style(theme.footer_text_style())
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()));
    frame.render_widget(footer, chunks[2]);
}
```

#### 演出状態 (`render_animating`)

同じ 3 分割レイアウト。コンテンツ部分に `capsule_machine::capsule_lines(anim_frame, theme)` を中央配置。フレーム 4-5 では `accent_alt` 色でフラッシュ。

下部に "turning... clack..." 等のテキスト表現を表示（フレームに応じて変化）:

- Frame 0-1: "turning..."
- Frame 2-3: "clack... clack..."
- Frame 4: "_ opening _"
- Frame 5: "OPENED!"

フッターは "[ ... ]" 表示（操作不可を示す）。
