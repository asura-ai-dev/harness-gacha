# Task 1007: カタログ画面描画と render ディスパッチ

## 概要

ホーム/カタログ画面の描画を `src/ui/catalog.rs` に実装し、`src/ui/render.rs` で画面ディスパッチを行う。TUI 起動後に pack 一覧が表示される状態にする。

## 詳細

### 1. `src/ui/render.rs`

画面ディスパッチを実装する。未実装の画面はプレースホルダー（画面名のみ表示）とする。

```rust
use ratatui::prelude::*;
use ratatui::widgets::Paragraph;
use crate::app::App;
use crate::screen::Screen;
use crate::ui::theme::cherry_cartridge;
use super::catalog;

pub fn render(frame: &mut Frame, app: &App) {
    let theme = cherry_cartridge();

    // 全画面共通の背景色を設定
    let bg = ratatui::widgets::Block::default()
        .style(Style::default().bg(theme.primary_bg));
    frame.render_widget(bg, frame.area());

    match app.current_screen {
        Screen::Catalog => catalog::render(frame, app, &theme),
        // 未実装画面はプレースホルダー
        other => {
            let text = format!("{:?} - Coming Soon", other);
            let p = Paragraph::new(text)
                .style(theme.text_style())
                .alignment(Alignment::Center);
            frame.render_widget(p, frame.area());
        }
    }
}
```

### 2. `src/ui/catalog.rs`

`agent-docs/ui-components.md` の画面 1（ホーム/カタログ）に従い実装する。

レイアウト分割（垂直 4 分割）:

1. タイトルバー: 3 行固定 (`Constraint::Length(3)`)
2. タブ/タグ行: 3 行固定 (`Constraint::Length(3)`)
3. Pack 一覧: 残り全部 (`Constraint::Fill(1)`)
4. フッター: 3 行固定 (`Constraint::Length(3)`)

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, List, ListItem, ListState, Paragraph, Tabs};
use crate::app::{App, CatalogTab};
use crate::ui::theme::Theme;
use crate::data::catalog as catalog_data;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),  // タイトル
            Constraint::Length(3),  // タブ
            Constraint::Fill(1),   // 一覧
            Constraint::Length(3),  // フッター
        ])
        .split(frame.area());

    render_title_bar(frame, chunks[0], app, theme);
    render_tab_bar(frame, chunks[1], app, theme);
    render_pack_list(frame, chunks[2], app, theme);
    render_footer(frame, chunks[3], theme);
}
```

#### タイトルバー

```rust
fn render_title_bar(frame: &mut Frame, area: Rect, app: &App, theme: &Theme) {
    // "harness-gacha" を左寄せ、検索クエリ表示を右寄せ
    // Block で枠線、border_style に theme.border
    let title = if app.search_active {
        format!("harness-gacha                    Search: [{}]", app.search_query)
    } else {
        "harness-gacha".to_string()
    };
    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(title)
        .style(theme.text_style())
        .block(block);
    frame.render_widget(p, area);
}
```

#### タブバー

```rust
fn render_tab_bar(frame: &mut Frame, area: Rect, app: &App, theme: &Theme) {
    let tab_titles = vec!["Featured", "Recent", "Recommended"];
    let selected = match app.catalog_state.current_tab {
        CatalogTab::Featured => 0,
        CatalogTab::Recent => 1,
        CatalogTab::Recommended => 2,
    };
    let tabs = Tabs::new(tab_titles)
        .select(selected)
        .style(theme.secondary_style())
        .highlight_style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(Block::default().borders(Borders::BOTTOM).border_style(theme.border_style()));
    frame.render_widget(tabs, area);
}
```

#### Pack 一覧

```rust
fn render_pack_list(frame: &mut Frame, area: Rect, app: &App, theme: &Theme) {
    let items: Vec<ListItem> = app.catalog_state.filtered_ids.iter()
        .filter_map(|id| catalog_data::find_pack_by_id(&app.catalog, id))
        .map(|pack| {
            // 各行: "  {name}    {targets}    YEN {price}    {trust_label}"
            let targets_str = pack.targets.iter()
                .map(|t| t.tool.as_str())
                .collect::<Vec<_>>()
                .join(" / ");
            let trust_label = if pack.permissions.has_danger() { "Risk: High" } else { "Safe: Low" };
            let line = format!(
                "  {}    {}    YEN {}    {}",
                pack.name, targets_str, pack.price, trust_label
            );
            // 購入済み判定は entitlements がある場合のみ（Phase 4 で接続）
            ListItem::new(line).style(theme.text_style())
        })
        .collect();

    let list = List::new(items)
        .block(Block::default().borders(Borders::LEFT | Borders::RIGHT).border_style(theme.border_style()))
        .highlight_style(theme.highlight_style())
        .highlight_symbol("> ");

    let mut state = ListState::default();
    state.select(Some(app.catalog_state.selected_index));
    frame.render_stateful_widget(list, area, &mut state);
}
```

}
