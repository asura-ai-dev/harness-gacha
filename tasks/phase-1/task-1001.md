# Task 1001: プロジェクト初期化と Cargo.toml 設定

## 概要

Rust プロジェクトを初期化し、全フェーズで使用する依存クレートを Cargo.toml に定義する。ディレクトリ構造の骨格を作成する。

## 詳細

### 1. Cargo.toml の作成

プロジェクトルート `/Users/kzt/Desktop/project-d/product/harness-gacha/Cargo.toml` を作成する。

```toml
[package]
name = "harness-gacha"
version = "0.1.0"
edition = "2021"

[dependencies]
ratatui = "0.29"
crossterm = "0.28"
serde = { version = "1", features = ["derive"] }
serde_json = "1"
chrono = { version = "0.4", features = ["serde"] }
rand = "0.8"
open = "5"
qrcode = "0.14"
arboard = "3"
thiserror = "2"
```

バージョンは `cargo add` 実行時の最新安定版に合わせて調整してよい。ただし ratatui は 0.29 以上、crossterm は 0.28 以上を使うこと。

### 2. ディレクトリ構造の作成

以下のディレクトリとファイルを作成する。中身はモジュール宣言のみの最小限でよい。

