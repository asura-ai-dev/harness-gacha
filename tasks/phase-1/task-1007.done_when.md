# Task 1007 done_when

- `cargo run` で TUI が起動し、カタログ画面が表示される
- タイトルバーに "harness-gacha" が表示される
- タブバーに "Featured", "Recent", "Recommended" の 3 タブが表示される
- `data/catalog.json` のサンプル pack がリスト形式で表示される（pack 名、対応ツール、価格、trust ラベルを含む）
- `j`/`Down` と `k`/`Up` キーでリスト内の選択行が移動する
- 選択行がハイライト表示される（accent 色の反転）
- `Tab` キーでタブが Featured -> Recent -> Recommended と切り替わる
- `BackTab` キーで逆方向にタブが切り替わる
- フッターにキーバインドのヘルプテキストが表示される
- `Enter` キーで選択中の pack の id が `selected_pack_id` に設定され、PackDetail 画面に遷移する（PackDetail 画面はプレースホルダー表示でよい）
- `b` または `Esc` で PackDetail プレースホルダーからカタログ画面に戻れる
- 背景色が Cherry Cartridge の `primary_bg`（#151018）で描画される
