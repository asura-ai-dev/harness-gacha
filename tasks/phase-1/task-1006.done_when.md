# Task 1006 done_when

- `cargo build` が成功する
- `cargo run` で TUI が起動し、alternate screen に切り替わる
- `q` キーまたは `Ctrl+C` で TUI が終了し、ターミナルが元の状態に復元される
- `src/app.rs` に `App` struct が存在し、`new()`, `navigate_to()`, `go_back()`, `update()`, `refresh_filtered_ids()` メソッドが実装されている
- `App::new()` が `data/catalog.json` を読み込んでカタログデータを保持する
- `navigate_to()` で画面遷移すると `screen_stack` に前の画面が push される
- `go_back()` で `screen_stack` から pop して前の画面に戻る
- `src/event.rs` に `poll_action()` 関数が存在し、crossterm イベントを Action に変換する
- `j`/`Down` で `Action::Down`、`k`/`Up` で `Action::Up`、`Enter` で `Action::Enter` が返る
- `src/main.rs` がイベントループを実行し、`enable_raw_mode` / `disable_raw_mode` でターミナルを管理する
