# Task 1002: データモデル定義（models/）

## 概要

全画面で使用するデータ構造体を `src/models/` 配下に定義する。serde の Serialize/Deserialize derive を付与し、JSON との相互変換を可能にする。

## 詳細

### 1. `src/models/permissions.rs`

`agent-docs/data-models.md` の Permissions セクションに従い、以下を実装する。

```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub struct Permissions {
    pub shell: bool,
    pub network: bool,
    pub filesystem_read: bool,
    pub filesystem_write: bool,
    pub git: bool,
}

impl Permissions {
    pub fn has_danger(&self) -> bool {
        self.shell || self.network
    }

    pub fn enabled_list(&self) -> Vec<&str> {
        let mut list = Vec::new();
        if self.shell { list.push("shell"); }
        if self.network { list.push("network"); }
        if self.filesystem_read { list.push("filesystem_read"); }
        if self.filesystem_write { list.push("filesystem_write"); }
        if self.git { list.push("git"); }
        list
    }
}
```

### 2. `src/models/catalog_entry.rs`

以下の struct をすべてこのファイルに定義する。

- `Author` - name (String), url (Option<String>), email (Option<String>)
- `Target` - tool (String), version_range (String)
- `ContentsSummary` - skills (u32), hooks (u32), templates (u32), other (u32)。全フィールドに `#[serde(default)]`
- `InstallInfo` - method (String), entrypoint (Option<String>), steps (Option<Vec<String>>)
- `LicenseInfo` - license_type (String, `#[serde(rename = "type")]`), text_url (Option<String>), spdx (Option<String>)
- `CatalogEntry` - `agent-docs/data-models.md` の CatalogEntry セクション通り

全 struct に `#[derive(Debug, Clone, Serialize, Deserialize)]` を付与する。
Optional フィールドには `#[serde(skip_serializing_if = "Option::is_none")]` を付与する。

### 3. `src/models/entitlement.rs`

- `EntitlementEntry` - pack_id, purchased_at, version_at_purchase, status, installed (bool), installed_version (Option<String>)
- `EntitlementStore` - user_id (String), entitlements (Vec<EntitlementEntry>)

### 4. `src/models/accounting.rs`

- `Transaction` - pack_id, user_id, amount (i64), tx_type (String, `#[serde(rename = "type")]`), timestamp, stripe_payment_id (Option<String>)
- `CreatorShare` - share_rate (f64), packs (Vec<String>)
- `AccountingData` - transactions (Vec<Transaction>), creator_shares (HashMap<String, CreatorShare>)

`use std::collections::HashMap;` を追加する。

### 5. `src/models/manifest.rs`

- `Manifest` - `agent-docs/data-models.md` の Manifest セクション通り
- Permissions, Author, Target, InstallInfo, LicenseInfo は `catalog_entry.rs` から re-export して使う

`src/models/manifest.rs` の先頭で:

```rust
use super::catalog_entry::{Author, Target, InstallInfo, LicenseInfo};
use super::permissions::Permissions;
```

### 6. `src/models/mod.rs` の更新

全子モジュールを pub mod で公開し、主要な型を re-export する。

```rust
pub mod catalog_entry;
pub mod entitlement;
pub mod accounting;
pub mod manifest;
pub mod permissions;

pub use catalog_entry::CatalogEntry;
pub use entitlement::{EntitlementEntry, EntitlementStore};
pub use accounting::{AccountingData, Transaction, CreatorShare};
pub use manifest::Manifest;
pub use permissions::Permissions;
```
