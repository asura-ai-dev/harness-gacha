# Task 2001: Pack 詳細画面

## 概要

カタログで選択した pack の詳細情報を表示する画面を `src/ui/pack_detail.rs` に実装する。F-3（Pack 詳細表示）に対応。

## 詳細

### 1. `src/ui/pack_detail.rs`

`agent-docs/ui-components.md` の画面 3（Pack 詳細）に従い実装する。

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 4 行固定 - pack 名、価格、creator、更新日
2. コンテンツ: 残り全部 - summary、targets、included summary、permission summary、sample preview
3. フッター: 3 行固定 - キーバインド

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Wrap};
use crate::app::App;
use crate::ui::theme::Theme;
use crate::data::catalog::find_pack_by_id;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    let pack = match &app.selected_pack_id {
        Some(id) => find_pack_by_id(&app.catalog, id),
        None => None,
    };
    let pack = match pack {
        Some(p) => p,
        None => {
            // pack が見つからない場合のフォールバック
            let msg = Paragraph::new("Pack not found")
                .style(theme.text_style())
                .alignment(Alignment::Center);
            frame.render_widget(msg, frame.area());
            return;
        }
    };

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(4),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    render_header(frame, chunks[0], pack, theme);
    render_content(frame, chunks[1], pack, app.scroll_offset, theme);
    render_footer(frame, chunks[2], theme);
}
```

#### ヘッダー

pack 名を左寄せ、"YEN {price}" を右寄せで 1 行目に表示。2 行目に "Creator: {author.name}" と "Updated: {updated_at}" を表示。Block でパネル背景。

```rust
fn render_header(frame: &mut Frame, area: Rect, pack: &CatalogEntry, theme: &Theme) {
    let line1 = Line::from(vec![
        Span::styled(&pack.name, Style::default().fg(theme.accent).add_modifier(Modifier::BOLD)),
        Span::raw("  "),
        Span::styled(format!("YEN {}", pack.price), Style::default().fg(theme.accent_alt)),
    ]);
    let line2 = Line::from(vec![
        Span::styled(format!("Creator: {}", pack.author.name), Style::default().fg(theme.text_secondary)),
        Span::raw("    "),
        Span::styled(format!("Updated: {}", pack.updated_at), Style::default().fg(theme.text_secondary)),
    ]);
    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(vec![line1, line2]).block(block);
    frame.render_widget(p, area);
}
```

#### コンテンツ

以下のセクションを縦に並べる。`scroll_offset` で上方向にスクロール可能にする。`Paragraph` の `scroll((app.scroll_offset, 0))` を使用。

1. **Summary** - `pack.summary` を表示。`pack.description` があれば追加
2. **Targets** - `pack.targets` を "- {tool} {version_range}" のリスト形式
3. **Included Summary** - `pack.contents_summary` を "skills: {n} hooks: {n} templates: {n}" 形式
4. **Permission Summary** - 各 permission を色分けして表示。shell/network が true なら `theme.danger` 色、filesystem_write/git が true なら `theme.warning` 色、disabled なら `theme.success` 色。`permission_span` ヘルパーを使う（task-2002 で実装するため、この時点ではインライン実装でよい）
5. **Sample Preview** - `pack.sample_preview` があれば引用形式で表示

各セクションのヘッダーは `theme.accent` 色で BOLD。

```rust
fn render_content(frame: &mut Frame, area: Rect, pack: &CatalogEntry, scroll: u16, theme: &Theme) {
    let mut lines: Vec<Line> = Vec::new();

    // Summary
    lines.push(Line::from(Span::styled("Summary", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    lines.push(Line::from(Span::styled(&pack.summary, Style::default().fg(theme.text_primary))));
    if let Some(desc) = &pack.description {
        lines.push(Line::from(Span::styled(desc.as_str(), Style::default().fg(theme.text_secondary))));
    }
    lines.push(Line::from(""));

    // Targets
    lines.push(Line::from(Span::styled("Targets", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    for t in &pack.targets {
        lines.push(Line::from(Span::styled(
            format!("  - {} {}", t.tool, t.version_range),
            Style::default().fg(theme.text_primary),
        )));
    }
    lines.push(Line::from(""));

    // Included Summary
    lines.push(Line::from(Span::styled("Included Summary", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    let cs = &pack.contents_summary;
    lines.push(Line::from(Span::styled(
        format!("  skills: {}  hooks: {}  templates: {}  other: {}", cs.skills, cs.hooks, cs.templates, cs.other),
        Style::default().fg(theme.text_primary),
    )));
    lines.push(Line::from(""));

    // Permission Summary
    lines.push(Line::from(Span::styled("Permission Summary", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
    let perms = &pack.permissions;
    let perm_items = [
        ("shell", perms.shell),
        ("network", perms.network),
        ("filesystem_read", perms.filesystem_read),
        ("filesystem_write", perms.filesystem_write),
        ("git", perms.git),
    ];
    for (name, enabled) in &perm_items {
        let (label, color) = if !enabled {
            ("no", theme.success)
        } else {
            match *name {
                "shell" | "network" => ("YES", theme.danger),
                "filesystem_write" | "git" => ("yes", theme.warning),
                _ => ("yes", theme.text_secondary),
            }
        };
        lines.push(Line::from(Span::styled(
            format!("  {}: {}", name, label),
            Style::default().fg(color),
        )));
    }
    lines.push(Line::from(""));

    // Sample Preview
    if let Some(sample) = &pack.sample_preview {
        lines.push(Line::from(Span::styled("Sample Preview", Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))));
        lines.push(Line::from(Span::styled(
            format!("  \"{}\"", sample),
            Style::default().fg(theme.text_secondary).add_modifier(Modifier::ITALIC),
        )));
    }

    let block = Block::default()
        .borders(Borders::LEFT | Borders::RIGHT)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(lines)
        .block(block)
        .wrap(Wrap { trim: false })
        .scroll((scroll, 0));
    frame.render_widget(p, area);
}
```
