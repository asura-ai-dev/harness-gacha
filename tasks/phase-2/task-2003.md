# Task 2003: 安全性詳細画面

## 概要

pack の権限と影響範囲を詳細表示する安全性詳細画面を `src/ui/safety_detail.rs` に実装する。F-4（安全性詳細表示）に対応。

## 詳細

### 1. `src/ui/safety_detail.rs`

`agent-docs/ui-components.md` の画面 4（安全性詳細）に従い実装する。

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 3 行固定 - "Safety Details: {pack_name}"
2. コンテンツ: 残り全部 - permissions、risks、review notes
3. フッター: 3 行固定 - キーバインド

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Wrap};
use crate::app::App;
use crate::ui::theme::Theme;
use crate::ui::widgets::permission_badge;
use crate::data::catalog::find_pack_by_id;

pub fn render(frame: &mut Frame, app: &App, theme: &Theme) {
    let pack = match &app.selected_pack_id {
        Some(id) => find_pack_by_id(&app.catalog, id),
        None => None,
    };
    let pack = match pack {
        Some(p) => p,
        None => {
            let msg = Paragraph::new("Pack not found")
                .style(theme.text_style())
                .alignment(Alignment::Center);
            frame.render_widget(msg, frame.area());
            return;
        }
    };

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    render_header(frame, chunks[0], pack, theme);
    render_content(frame, chunks[1], pack, theme);
    render_footer(frame, chunks[2], theme);
}
```

#### ヘッダー

```rust
fn render_header(frame: &mut Frame, area: Rect, pack: &CatalogEntry, theme: &Theme) {
    let title = format!("Safety Details: {}", pack.name);
    let block = Block::default()
        .borders(Borders::ALL)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(title)
        .style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(block);
    frame.render_widget(p, area);
}
```

#### コンテンツ

3 セクションを縦に並べる:

1. **Permissions** - 5 つの権限を `permission_badge::permission_line()` で表示。各行に permission 名と enabled/disabled を色付きで表示。danger 権限（shell, network）が enabled の場合は BOLD + danger 色で目立たせる
2. **Risks** - `pack.risks` がある場合、リスト形式で表示。`theme.warning` 色で "- {risk}" 形式
3. **Review Notes** - `pack.review_notes` がある場合、`theme.text_secondary` 色で表示

```rust
fn render_content(frame: &mut Frame, area: Rect, pack: &CatalogEntry, theme: &Theme) {
    let mut lines: Vec<Line> = Vec::new();

    // Permissions セクション
    lines.push(Line::from(Span::styled(
        "Permissions",
        Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
    )));
    lines.push(Line::from(""));

    let perms = [
        ("shell", pack.permissions.shell),
        ("network", pack.permissions.network),
        ("filesystem_read", pack.permissions.filesystem_read),
        ("filesystem_write", pack.permissions.filesystem_write),
        ("git", pack.permissions.git),
    ];
    for (name, enabled) in &perms {
        lines.push(permission_badge::permission_line(name, *enabled, theme));
    }
    lines.push(Line::from(""));

    // Risks セクション
    if let Some(risks) = &pack.risks {
        if !risks.is_empty() {
            lines.push(Line::from(Span::styled(
                "Risks",
                Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
            )));
            lines.push(Line::from(""));
            for risk in risks {
                lines.push(Line::from(Span::styled(
                    format!("  - {}", risk),
                    Style::default().fg(theme.warning),
                )));
            }
            lines.push(Line::from(""));
        }
    }

    // Review Notes セクション
    if let Some(notes) = &pack.review_notes {
        lines.push(Line::from(Span::styled(
            "Review Notes",
            Style::default().fg(theme.accent).add_modifier(Modifier::BOLD),
        )));
        lines.push(Line::from(""));
        lines.push(Line::from(Span::styled(
            format!("  {}", notes),
            Style::default().fg(theme.text_secondary),
        )));
    }

    let block = Block::default()
        .borders(Borders::LEFT | Borders::RIGHT)
        .border_style(theme.border_style())
        .style(theme.panel_style());
    let p = Paragraph::new(lines)
        .block(block)
        .wrap(Wrap { trim: false });
    frame.render_widget(p, area);
}
```
