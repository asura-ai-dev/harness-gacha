# Task 2003 done_when

- Pack 詳細画面で `s` キーを押すと安全性詳細画面に遷移する
- 安全性詳細画面のヘッダーに "Safety Details: {pack名}" が表示される
- Permissions セクションに shell, network, filesystem_read, filesystem_write, git の 5 項目が表示される
- shell=true の pack で "shell" 行が danger 色（赤系）かつ BOLD で表示される
- network=true の pack で "network" 行が danger 色（赤系）かつ BOLD で表示される
- filesystem_write=true の pack で "filesystem_write" 行が warning 色（黄系）で表示される
- disabled の権限が success 色（緑系）で "disabled" と表示される
- risks フィールドがある pack で Risks セクションが表示される
- review_notes フィールドがある pack で Review Notes セクションが表示される
- `p` キーで購入案内画面に遷移する（プレースホルダー表示でよい）
- `b` または `Esc` で Pack 詳細画面に戻る
- カタログ -> Pack 詳細 -> 安全性詳細 -> `b` -> Pack 詳細 -> `b` -> カタログ の遷移が正しく動作する
