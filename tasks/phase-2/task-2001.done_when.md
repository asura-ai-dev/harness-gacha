# Task 2001 done_when

- `cargo run` でカタログ画面から pack を選択して `Enter` を押すと Pack 詳細画面に遷移する
- Pack 詳細画面に pack 名と価格（YEN 表記）がヘッダーに表示される
- Creator 名と更新日がヘッダーに表示される
- Summary セクションに pack.summary の内容が表示される
- Targets セクションに対応ツール名とバージョン範囲がリスト形式で表示される
- Included Summary セクションに skills, hooks, templates, other の件数が表示される
- Permission Summary セクションに 5 つの権限（shell, network, filesystem_read, filesystem_write, git）が表示される
- shell=true または network=true の pack で該当権限が danger 色（赤系）で表示される
- disabled の権限が success 色（緑系）で表示される
- sample_preview がある pack でサンプルテキストが引用形式で表示される
- `j`/`Down` と `k`/`Up` でコンテンツがスクロールする
- `s` キーで SafetyDetail 画面に遷移する（プレースホルダー表示でよい）
- `p` キーで Purchase 画面に遷移する（プレースホルダー表示でよい）
- `b` または `Esc` でカタログ画面に戻る
- selected_pack_id が None の場合に "Pack not found" が表示されクラッシュしない
