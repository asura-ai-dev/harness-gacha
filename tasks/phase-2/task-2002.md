# Task 2002: 権限バッジ Widget

## 概要

permission の enabled/disabled を色付きで表示するヘルパー関数群を `src/ui/widgets/permission_badge.rs` に実装する。Pack 詳細画面と安全性詳細画面の両方から使用する。

## 詳細

### 1. `src/ui/widgets/permission_badge.rs`

```rust
use ratatui::prelude::*;
use ratatui::text::Span;
use crate::ui::theme::Theme;

/// 単一の permission を色付き Span として返す
/// - shell, network が enabled → danger 色
/// - filesystem_write, git が enabled → warning 色
/// - filesystem_read が enabled → text_secondary 色
/// - disabled → success 色
pub fn permission_span<'a>(name: &'a str, enabled: bool, theme: &Theme) -> Span<'a> {
    let (label, color) = if !enabled {
        ("disabled", theme.success)
    } else {
        match name {
            "shell" | "network" => ("ENABLED", theme.danger),
            "filesystem_write" | "git" => ("enabled", theme.warning),
            _ => ("enabled", theme.text_secondary),
        }
    };
    Span::styled(
        format!("{}: {}", name, label),
        Style::default().fg(color),
    )
}

/// 安全性詳細画面用: permission 行を Line として返す（左にラベル、右に状態）
/// danger の場合は BOLD modifier も付与
pub fn permission_line<'a>(name: &'a str, enabled: bool, theme: &Theme) -> Line<'a> {
    let (label, color, bold) = if !enabled {
        ("disabled", theme.success, false)
    } else {
        match name {
            "shell" | "network" => ("ENABLED", theme.danger, true),
            "filesystem_write" | "git" => ("enabled", theme.warning, false),
            _ => ("enabled", theme.text_secondary, false),
        }
    };

    let mut style = Style::default().fg(color);
    if bold {
        style = style.add_modifier(Modifier::BOLD);
    }

    // 固定幅でアラインメント
    Line::from(vec![
        Span::styled(format!("  {:<20}", name), Style::default().fg(theme.text_primary)),
        Span::styled(label.to_string(), style),
    ])
}

/// 権限サマリを 1 行で返す（カタログ一覧やカード表示用）
/// 例: "shell / write あり, network なし"
pub fn permission_summary_text(
    shell: bool,
    network: bool,
    filesystem_write: bool,
    git: bool,
    theme: &Theme,
) -> Vec<Span<'static>> {
    let mut parts: Vec<Span<'static>> = Vec::new();
    let mut dangerous: Vec<&str> = Vec::new();
    let mut safe: Vec<&str> = Vec::new();

    if shell { dangerous.push("shell"); } else { safe.push("shell"); }
    if network { dangerous.push("network"); } else { safe.push("network"); }
    if filesystem_write { dangerous.push("write"); }
    if git { dangerous.push("git"); }

    if !dangerous.is_empty() {
        parts.push(Span::styled(
            dangerous.join(" / ") + " あり",
            Style::default().fg(theme.danger),
        ));
    }
    if !dangerous.is_empty() && !safe.is_empty() {
        parts.push(Span::raw(", "));
    }
    if !safe.is_empty() && !dangerous.is_empty() {
        // safe 側は network なしなど主要なもののみ表示
        parts.push(Span::styled(
            safe.join(" / ") + " なし",
            Style::default().fg(theme.success),
        ));
    }
    parts
}
```
