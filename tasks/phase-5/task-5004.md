# Task 5004: エラーハンドリング整備

## 概要

アプリケーション全体のエラーハンドリングを整備する。panic 防止、graceful なエラー表示、ターミナル復元の保証を実装する。

## 詳細

### 1. カスタムエラー型の定義

`src/` 直下に `error.rs` を新規作成する。

```rust
use thiserror::Error;

#[derive(Error, Debug)]
pub enum AppError {
    #[error("データ読み込みエラー: {0}")]
    DataLoad(String),

    #[error("JSON パースエラー: {0}")]
    JsonParse(#[from] serde_json::Error),

    #[error("ファイル I/O エラー: {0}")]
    Io(#[from] std::io::Error),

    #[error("ブラウザ起動エラー: {0}")]
    BrowserOpen(String),

    #[error("クリップボードエラー: {0}")]
    Clipboard(String),
}
```

`src/main.rs` に `mod error;` を追加する。

### 2. panic 時のターミナル復元

`src/main.rs` にカスタム panic hook を設定する。panic 発生時に raw mode を解除し、alternate screen を離脱してからデフォルトの panic handler を呼ぶ。

```rust
fn install_panic_hook() {
    let original_hook = std::panic::take_hook();
    std::panic::set_hook(Box::new(move |panic_info| {
        // ターミナルを復元
        let _ = crossterm::terminal::disable_raw_mode();
        let _ = crossterm::execute!(
            std::io::stdout(),
            crossterm::terminal::LeaveAlternateScreen
        );
        // デフォルトの panic 出力
        original_hook(panic_info);
    }));
}
```

`main()` の先頭で `install_panic_hook()` を呼ぶ。

### 3. main 関数の Result 化

`main()` を `Result<(), Box<dyn std::error::Error>>` を返すように変更し、`?` 演算子でエラーを伝播する。エラー時もターミナル復元を保証する。

```rust
fn main() -> Result<(), Box<dyn std::error::Error>> {
    install_panic_hook();

    enable_raw_mode()?;
    let mut stdout = io::stdout();
    execute!(stdout, EnterAlternateScreen)?;
    let backend = CrosstermBackend::new(stdout);
    let mut terminal = Terminal::new(backend)?;

    let result = run_app(&mut terminal);

    // 必ず復元する
    disable_raw_mode()?;
    execute!(terminal.backend_mut(), LeaveAlternateScreen)?;
    terminal.show_cursor()?;

    result
}

fn run_app(terminal: &mut Terminal<CrosstermBackend<io::Stdout>>) -> Result<(), Box<dyn std::error::Error>> {
    let data_dir = std::path::Path::new("data");
    let mut app = app::App::new(data_dir);

    let tick_rate = Duration::from_millis(250);
    while app.running {
        terminal.draw(|frame| ui::render::render(frame, &app))?;
        let action = event::poll_action(tick_rate, app.current_screen, app.search_active);
        app.update(action);
    }
    Ok(())
}
```
