# Task 5002: ヘルプ / 法務情報画面

## 概要

キーバインドヘルプと法務表示（特商法表示への導線、利用規約、返金ポリシー、お問い合わせ先）を表示する画面を `src/ui/help.rs` に実装する。

## 詳細

### 1. `src/ui/help.rs`

`agent-docs/ui-components.md` の画面 8（ヘルプ / 法務情報）に従い実装する。

レイアウト分割（垂直 3 分割）:

1. ヘッダー: 3 行固定 - "Help"
2. コンテンツ: 残り全部 - キーバインド一覧 + 法務情報
3. フッター: 3 行固定 - キーバインド

```rust
use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Wrap};
use crate::app::App;
use crate::ui::theme::Theme;

pub fn render(frame: &mut Frame, _app: &App, theme: &Theme) {
    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .constraints([
            Constraint::Length(3),
            Constraint::Fill(1),
            Constraint::Length(3),
        ])
        .split(frame.area());

    // ヘッダー
    let header = Paragraph::new("Help")
        .style(Style::default().fg(theme.accent).add_modifier(Modifier::BOLD))
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()).style(theme.panel_style()));
    frame.render_widget(header, chunks[0]);

    // コンテンツ
    render_content(frame, chunks[1], theme);

    // フッター
    let footer = Paragraph::new("[b] 戻る")
        .style(theme.footer_text_style())
        .block(Block::default().borders(Borders::ALL).border_style(theme.border_style()));
    frame.render_widget(footer, chunks[2]);
}
```
