# harness-gacha

Claude Code / Codex 向けの harness pack を扱う、キュレーション型ストア構想のリポジトリです。

初期方針は以下です。

- オープン marketplace ではなく、キュレーション型の `再販モデル` で始める
- ガチャ風の体験は無料の discovery に限定し、購入は固定価格にする
- 決済は Web checkout に逃がし、TUI は discovery、信頼性表示、導入体験に集中させる
- 販売後にクリエイターへ売上分配を行う

- `docs/mvp-prd.md`: MVP 要件定義
- `docs/manifest-spec.md`: `manifest.json` の仕様書
