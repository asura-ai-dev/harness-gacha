# 画面ワイヤー

## 目的

この文書は、`harness-gacha` の MVP に必要な TUI 画面をテキストベースで可視化するためのワイヤー集である。実装前に、どの情報をどこに置くか、どこまで見せるか、どこで購入導線に送るかを合わせることを目的とする。

## 前提

- TUI は discovery、比較、信頼判断、導入支援に集中する
- 決済入力は TUI 内で行わず、Web checkout に遷移する
- 購入前は安心して判断できる情報を見せる
- 購入前は pack を自作できるほどの情報は見せない

## 3. Pack 詳細

役割:
- 購入判断の中心画面

```text
+----------------------------------------------------------------------------------+
| Team Review Pack                                              YEN 1,800          |
| Creator: Example Creator                    Updated: 2026-04-10                  |
+----------------------------------------------------------------------------------+
| Summary                                                                          |
| コードレビュー、handoff、pre-commit 補助をまとめた team 向け pack               |
|                                                                                  |
| Targets                                                                          |
| - Claude Code >= 1.0.0                                                           |
| - Codex >= 0.9.0                                                                 |
|                                                                                  |
| Included Summary                                                                 |
| - skills: 3                                                                      |
| - hooks: 2                                                                       |
| - templates: 4                                                                   |
|                                                                                  |
| Permission Summary                                                               |
| - shell: yes   - network: no   - fs_write: yes   - git: no                      |
|                                                                                  |
| Sample Preview                                                                   |
| "PR review の際は、変更の意図と回帰リスクを先に..."                             |
|                                                                                  |
+----------------------------------------------------------------------------------+
| [S] 安全性詳細   [P] 購入案内   [B] 戻る                                         |
```

表示要素:
- summary
- creator
- targets
- 同梱カテゴリ件数
- 権限サマリ
- 短い抜粋

表示しないもの:
- prompt 全文
- hook 全文
- agent instruction 全文
- template 完全版
- install script の詳細実装

## 4. 安全性詳細

役割:
- 権限と影響範囲を明示する

```text
+----------------------------------------------------------------------------------+
| Safety Details: Team Review Pack                                                 |
+----------------------------------------------------------------------------------+
| Permissions                                                                      |
| - shell: enabled                                                                 |
| - network: disabled                                                              |
| - filesystem_read: enabled                                                       |
| - filesystem_write: enabled                                                      |
| - git: disabled                                                                  |
|                                                                                  |
| Impact                                                                           |
| - 作業ディレクトリ配下のファイルを更新する可能性あり                             |
| - 外部 API 送信なし                                                              |
|                                                                                  |
| Review Notes                                                                     |
| - install script はローカルコピーのみ                                            |
| - credential 送信処理なし                                                        |
|                                                                                  |
+----------------------------------------------------------------------------------+
| [P] 購入案内   [B] 詳細へ戻る                                                    |
+----------------------------------------------------------------------------------+
```

表示要素:
- 権限一覧
- 想定影響
- review による補足

## 画面遷移サマリ

```text
ホーム / カタログ
  -> Discovery
  -> Pack 詳細
  -> 購入済みライブラリ

Discovery
  -> 待機状態
  -> 演出状態
  -> 結果状態
  -> Pack 詳細
  -> ホーム / カタログ

Pack 詳細
  -> 安全性詳細
  -> 購入案内
  -> ホーム / カタログ

安全性詳細
  -> 購入案内
  -> Pack 詳細

購入案内
  -> Web checkout
  -> Pack 詳細

購入済みライブラリ
  -> インストール詳細

インストール詳細
  -> 購入済みライブラリ
```

## 購入前表示の境界

見せるもの:
- 価格
- creator
- targets
- 権限
- 外部通信の有無
- 同梱カテゴリ件数
- 短い抜粋
- 更新日

見せないもの:
- prompt 全文
- hook 全文
- agent instruction 全文
- template 完全版
- install script の詳細実装
- pack を再現できる粒度の内部ロジック

## ガチャガチャスクリーンの実装メモ

- TUI で本物の物理表現を再現する必要はない
- 記号、罫線、反転、色変化で十分にガチャガチャ感を出せる
- 重要なのは `待機 -> 演出 -> 結果` のテンポである
- 長いアニメーションより、短く気持ちよい反応を優先する
